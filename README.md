# sharing-post

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Json Server
db folder contains db.json file
```
json-server --watch db.json or npx json-server --watch db.json
```
